# Credit_Card_Finance_Dashboard
📌 Project Overview

This project analyzes 650K+ credit card transactions to uncover revenue trends, customer behavior patterns, and key business drivers.

An interactive dashboard was developed using Power BI to help stakeholders monitor KPIs, identify high-performing customer segments, and understand spending patterns across different dimensions.

🎯 Business Problem

- Financial institutions need visibility into:
- Revenue contribution by card category
- Spending behavior across transaction types
- Customer segmentation (education, job, gender)
- Quarterly revenue trends
- Interest earned from transactions

The objective was to build a dashboard that provides actionable insights for strategic decision-making.

📊 Key KPIs

- Total Revenue: ₹55M
- Total Transaction Amount: ₹45M
- Total Interest Earned: ₹7.84M
- Total Transactions: 656K+

  🛠 Tools & Technologies
  - Data Modeling
  - DAX Measures
  - Interactive Visualizations
  - Slicers & Filters

 🔍 Key Insights

- Blue card customers generate the majority of revenue.
- Bills and essential expenses contribute the highest transaction value.
- Graduate customers contribute the highest revenue among education segments.
- Swipe transactions dominate usage patterns compared to online transactions.

🚀 Business Impact

This dashboard enables:
✔ Revenue monitoring across time
✔ Identification of high-value customer segments
✔ Data-driven marketing & retention strategies
✔ Performance tracking of card categories
